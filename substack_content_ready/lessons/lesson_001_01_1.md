# Title: 1.1 Introducing Myself - My Name is- Mama Nama  
## Concepts: "Mama Nama"

## Learning
In Sanskrit, we say "Mama" for myself.  "Nama" is used for "Name". Masculine sounds usually end with the "с╕е" sound, so "Suresh" gets transformed into "Sure┼Ыaс╕е".  Examples have been given below on how to state your name using the "Mama Nama" prefix
