# Title: 19.3 Without - vin─Б - рд╡рд┐рдирд╛
## Concepts: "vin─Б - Without"

## Learning
"Vin─Б" means "without" in Sanskrit. It is a preposition used to indicate absence or lack. Understanding prepositions is essential for expressing relationships between words in Sanskrit.

| Sanskrit           | Transliteration      | Meaning                          |
| ------------------ | -------------------- | -------------------------------- |
| рд╡рд┐рдирд╛               | Vin─Б                 | Without                          |
| рд╡рд┐рдирд╛ рдЧрдЪреНрдЫрддрд┐       | Vin─Б gacchati        | Goes without                     |
| рд╡рд┐рдирд╛ рдкрдарддрд┐         | Vin─Б paс╣нhati         | Reads without                    |
| рд╡рд┐рдирд╛ рд▓рд┐рдЦрддрд┐        | Vin─Б likhati         | Writes without                   |
| рд╡рд┐рдирд╛ рдХрд░реЛрддрд┐        | Vin─Б karoti          | Does without                     |
